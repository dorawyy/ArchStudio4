Building ArchStudio Extensions

The first step to build ArchStudio extensions is to check out and build ArchStudio from source. After that, development proceeds by creating new components in the Myx style, or modifying existing components.

More documentation on how to extend ArchStudio is forthcoming. Until then, documentation on accessing specific aspects of ArchStudio are contained in the other 'Developing' sections of this site:

  • Using the xArchADT Component
  • Using the XArchFlatProxy interface

For more information, contact Eric Dashofy.
