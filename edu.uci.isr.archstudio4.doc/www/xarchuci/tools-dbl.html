xArch/xADL 2.0 Data Binding Library

We are extremely proud to release initial versions of the xArch/xADL 2.0 Data Binding Library for creating, parsing, manipulating, and writing xArch and xADL 2.0 documents. These libraries provide a convenient, xArch/xADL 2.0-oriented Java interface to xArch documents. Libraries currently available support the instance (core) xArch specification, as well as all the extensions on our extensions page. These libraries are available for immediate download.

For an example of how to use these libraries, click here.

Note! Previous versions of this library used Apache Xerces as the underlying XML parser. The latest version of the library has been converted to use the Sun Crimson parser included with J2SDK 1.4. As such, the library must be executed and run with J2SDK1.4.

You can download J2SDK1.4 here if you do not already have it.
