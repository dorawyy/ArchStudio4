Apigen

If you've used the xArch/xADL 2.0 Data Binding Library, you'll notice that many of the files are authored by "Apigen." Apigen is an XML schema-to-Java data binding generator for xArch schemas that automatically generates those libraries based only on the XML schemas. It supports a large subset of the XML schema language.

Apigen is still in an Alpha state, and full documentation on using the tool is not yet available. However, we have evolved it to the point of release. If you are involved in developing your own xArch/xADL 2.0 extension schemas, feel free to try them with apigen. However, please also contact Eric Dashofy for assistance.

Note! Previous versions of this product used Apache Xerces as the underlying XML parser. The latest version of the library has been converted to use the Sun Crimson parser included with J2SDK 1.4. As such, the product must be executed and run with J2SDK1.4.

You can download J2SDK1.4 here if you do not already have it.
