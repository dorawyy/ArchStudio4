xArch/xADL 2.0 Tools Overview

UCI has developed several tools to the help developers create, manage, and manipulate xArch documents and schemas. Our current set of tools include Java DOM-Based Libraries for creating, parsing, manipulating, and writing xArch documents using the instance (core) schema and any of the extensions we have listed on the extensions page. Furthermore, if you have created an extension of your own for xArch, we can help you in rapidly generating similar and compatible DOM-based Java Libraries for your extension.

Available Tools

  • xArch/xADL 2.0 Data Binding Library - Java libraries for creating and manipulating xArch and xADL 2.0 documents.
  • Apigen - Java tool to generate the Data Binding Library, above, using the xArch and xADL 2.0 schemas.
